This project demonstrates a highly available and secure 2-Tier AWS Architecture built using the AWS Management Console.

The architecture includes:

Custom VPC

Public and Private Subnets across two Availability Zones

Internet Gateway

NAT Gateway

Bastion Host

Application Load Balancer (ALB)

Target Groups & Health Checks

Auto Scaling Group (ASG)

Security Groups with controlled access

The application is deployed on EC2 instances in private subnets and accessed securely via an Application Load Balancer.
